Welcome to DelightMate

DelightMate is an AI assistant platform built with Flask. It provides voice interaction tools and AI agent capabilities.

Try Voice Interface

API Endpoints

Speech to Text

POST /api/stt

Convert audio to text using OpenAI Whisper.

Send a multipart/form-data request with an audio file.

Text to Speech

POST /api/tts

Convert text to speech using ElevenLabs.

Send a JSON payload with a "text" field.

AI Agent

POST /api/agent

Interact with the GPT-4 powered agent.

Send a JSON payload with a "transcript" field.
